# RescuEats: Smart Food Redistribution Platform

Welcome to the official repository of **RescuEats**, a platform dedicated to reducing food waste by connecting donors with charities.

## 🚀 Live Demo
You can view the live operational platform here:
[https://haithamqoqzaeh-ops.github.io/RescuEats/](https://haithamqoqzaeh-ops.github.io/RescuEats/)

## 💻 About the Project
This project was developed as part of my graduation requirements. It features:
- **Donor Interface:** For restaurants to post surplus food.
- **Charity Dashboard:** For organizations to claim and track donations.
- **Driver Management:** To handle efficient food delivery.
- **Admin Control:** For system oversight and moderation.

## 🛠 Technologies Used
- HTML5, CSS3, JavaScript
- GitHub Pages (Deployment)
